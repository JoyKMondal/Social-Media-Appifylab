# Social Media Clone

A modern full-stack social media application. Users can register, log in, create public or private posts with text and images, like/unlike posts & comments, add threaded comments/replies, and see who liked what — all while sticking 100% to the provided HTML/CSS designs.

### Built With

All the core technologies used in this project are included below:

### Tech Stack & Tools

- **Frontend**: React.js Vite — for optimal performance, and fast ui updates  
- **Styling**: Tailwind CSS, Daisy UI and inline plain CSS.  
- **Backend**: Node.js, Express.js, 
- **Authentication**: JWT stored in httpOnly cookies (secure, refresh-token ready)  
- **Database**: MongoDB with Mongoose Schema — type-safe, scalable, and migration-ready  
- **Image Uploads**: Cloudinary (direct client upload with signed presets)  
- **Validation**: Zod for both client and server-side form validation  
- **State Management**: Context API  
- **Deployment**: Frontend on Vercel and Backend on Render 
- **Other**: TypeScript, ESLint + Prettier, responsive design, loading states, error handling
